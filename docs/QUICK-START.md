## 🔧 调试快速定位

### 前端问题
1. **组件不显示** → 检查 Console 日志
2. **样式异常** → 检查 CSS 类名
3. **下载失败** → 检查 currentOutput

### 后端问题
1. **API 失败** → 检查 Supabase 日志
2. **生成失败** → 检查 VolcEngine 配置
3. **保存失败** → 检查 Supabase 连接
